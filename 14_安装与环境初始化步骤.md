## 4. 从零安装步骤（团队复现版）
### 第一步：安装 nvm
```bash
export NVM_DIR="$HOME/.nvm"
curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

### 第二步：安装 Node.js LTS
```bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"
nvm install --lts
nvm use --lts
node -v
npm -v
```

### 第三步：创建前端工程（Vite + TS）
```bash
cd /Users/xujunming/Desktop/KingHonor_GreatWall_AI_Training_Project
npm create vite@latest game-demo -- --template vanilla-ts
```

### 第四步：安装依赖与 Phaser
```bash

```

### 第五步：启动与构建验证

## 6. 常见问题
### 问题 A：`node: command not found`
原因：新终端未加载 `nvm`。

处理：先执行“第 3 节”的前置命令，再执行 `nvm use --lts`。

### 问题 B：`npm install` 网络慢或中断
处理建议：
- 重试一次安装命令
- 避免多个 npm 进程同时跑
- 保持同一终端会话，减少环境丢失

### 问题 C：`npm run dev` 启动后页面空白
处理建议：
- 确认终端无报错
- 刷新浏览器并看控制台错误
- 先跑 `npm run build` 确认 TypeScript 能通过
